# saucedemo-test

Este proyecto contiene pruebas automatizadas con Selenium y Pytest usando el patrón Page Object Model (POM).

#Requisitos
- Python 3.8+
- Google Chrome
- Chromedriver

Instalación
bash
pip install -r requirements.txt
